## `StarryOS` 简介

`StarryOS` 是一个基于 `ArceOS` 系统实现的宏内核的操作系统，使用 `Rust` 语言开发。

`Starry`意指布满星星✨的，寓意本`OS`的开发学习借鉴了许多的前辈的思路，并将其汇总归一为这个内核。主要开发成员包含：陈嘉钰、郑友捷、王昱栋。

`StarryOS` 是一个开源项目，欢迎大家参与贡献。
